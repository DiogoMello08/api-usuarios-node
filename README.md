🚀 API de Usuários

API REST de usuários com autenticação JWT desenvolvida em Node.js.

📌 Funcionalidades
- Cadastro de usuário
- Login com autenticação JWT
- Rota protegida

🛠️ Tecnologias
- Node.js
- Express
- Bcrypt
- JSON Web Token

🔐 Autenticação
A API utiliza JWT para proteger rotas e autenticar usuários.

⚠️ Observação
Atualmente os dados são armazenados em memória (sem banco de dados).

🚀 Como rodar o projeto

```bash
npm install
npm run dev
